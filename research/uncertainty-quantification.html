Research Overview

This body of work provides a comprehensive implementation layer for uncertainty quantification (UQ), designed to support modeling workflows where data, parameters, and structural assumptions all contribute to uncertainty. The project spans both theoretical methods and practical numerical implementations.

The portfolio covers six major areas: sensitivity analysis, parameter estimation, MCMC sampling, frequentist confidence and prediction intervals, surrogate modeling, and model discrepancy detection. Together, these components form a flexible toolkit for calibration, inference, propagation, and diagnostics.

Implementations were developed in both Python and MATLAB, making the repository useful for research prototyping, teaching, and reproducible computational experiments. The result is a broad, method-oriented resource rather than a single narrowly scoped application.

Core Contributions

  • Implemented local and global sensitivity analysis, including Morris screening and Sobol index estimation.
  • Covered parameter estimation workflows based on ordinary least squares and constrained optimization.
  • Included MCMC algorithms such as Metropolis-Hastings and Adaptive Metropolis for Bayesian calibration.
  • Added interval estimation, surrogate modeling, and model-discrepancy diagnostics in a unified repository.

Mathematical Formulation

Uncertainty quantification links data, models, and parameters through both deterministic and probabilistic reasoning. Bayesian inversion and variance-based sensitivity indices are two central examples that appear repeatedly throughout the project.

\[p(\theta\mid y) \propto p(y\mid \theta)\,p(\theta)\]Bayes’ theorem for posterior inference, the starting point for calibration and MCMC sampling.
\[S_i = \frac{\operatorname{Var}_{\theta_i}\!\left(\mathbb{E}[Y\mid \theta_i]\right)}{\operatorname{Var}(Y)}, \qquad S_{T_i} = 1 - \frac{\operatorname{Var}_{\theta_{\sim i}}\!\left(\mathbb{E}[Y\mid \theta_{\sim i}]\right)}{\operatorname{Var}(Y)}\]First-order and total-order Sobol indices measuring the contribution of a parameter to output variance.

Why it matters

By assembling the full UQ stack in one place, the project helps bridge theory and practice: researchers can move from parameter screening to inference, surrogate construction, and discrepancy analysis without changing conceptual frameworks.

Methodology

Sensitivity Analysis

The toolkit includes local derivative-based analyses together with global methods such as Morris screening and Sobol indices for ranking influential inputs and interactions.

Inference and Calibration

Parameter estimation combines least-squares optimization with Bayesian calibration via MCMC, supporting both deterministic fitting and posterior uncertainty analysis.

Surrogates and Diagnostics

Polynomial approximations and Gaussian Process surrogates reduce computational cost, while model discrepancy tools help diagnose structural mismatch between simulations and observed data.
