Interpretable Discovery of Hidden Dynamics in BK Virus Infection

NC State University · PhD Dissertation Research

The scientific goal was to fit serum-creatinine trajectories after BK virus infection and recover a concise, interpretable dynamical model. The workflow combines PINNs for trajectory fitting, causal inference for variable selection, and symbolic regression for extracting human-readable ODE terms.

Hidden ODE structure from clinical signals
PINNs + causal inference + symbolic regression
Interpretability treated as a primary outcome

Problem framing

Let \(c(t)\) denote serum creatinine and let \(u(t)\) summarize clinically relevant covariates or interventions. The hidden-dynamics problem assumes that observed trajectories are generated by an underlying differential mechanism

\[ \dot{c}(t) = g\big(c(t), u(t), t\big), \]

but that \(g\) is not directly known. The workflow used in this dissertation imposed three explicit assumptions:

  • trajectory data are noisy and incomplete, so flexible function approximation is necessary;
  • clinical interpretation requires more than prediction error, so latent dynamics must be made legible;
  • candidate mechanisms should be distilled into human-readable expressions that can support hypothesis generation.

Clinical setting

Renal transplantation with serum-creatinine changes following BK virus infection.

Interpretability target

Recover parsimonious governing terms instead of stopping at black-box trajectory fit.

Research outcome

A mechanistic narrative that links learned dynamics to inspectable equation structure.

Three-Stage Pipeline

1

PINNs

Fit noisy longitudinal trajectories while penalizing violations of the candidate dynamical law, so the learned representation remains scientifically constrained.

2

Causal inference

Separate potentially meaningful drivers from spurious correlations to decide which variables and interactions deserve mechanistic interpretation.

3

Symbolic regression

Translate the learned black-box response into compact analytical terms that can be inspected, compared, and communicated as candidate ODE structure.

Equations Behind the Workflow

Composite PINN loss

\[ \mathcal{L}_{\mathrm{PINN}} = \mathcal{L}_{\mathrm{data}} + \lambda_{\mathrm{phys}}\,\mathcal{L}_{\mathrm{physics}} \]

The first term fits measured trajectories, while the second penalizes disagreement with the governing differential constraint.

\[ \mathcal{L}_{\mathrm{physics}} = \frac{1}{N_r}\sum_{i=1}^{N_r}\left\|\dot{c}(t_i) - g_{\theta}\big(c(t_i), u(t_i), t_i\big)\right\|^2 \]

Interpretability objective

\[ \hat{g} = \arg\min_{h \in \mathcal{H}} \left\{ \sum_{i=1}^{N} \left|\dot{c}(t_i) - h\big(c(t_i), u(t_i), t_i\big)\right|^2 + \eta\,\Omega(h) \right\} \]

Symbolic regression searches a hypothesis class \(\mathcal{H}\) for a sparse expression whose dynamics match the learned trajectory derivative while keeping complexity controlled by \(\Omega(h)\).
