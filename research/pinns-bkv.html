Interpretable Discovery of Hidden Dynamics in BK Virus Infection

NC State University · PhD Dissertation Research

BK virus nephropathy is a common cause of kidney transplant loss, and no effective drugs specifically target BKV. Doctors must balance immunosuppressive medication to prevent rejection versus infection. This project developed an integrated PINNs + causal inference + symbolic regression framework to model serum creatinine dynamics and extract interpretable mathematical expressions for the missing physics.

EHR from Duke Hospitals · 900+ patients · 493 with BKV dynamics
PINNs + Causal Inference + Symbolic Regression pipeline
Goal: model serum creatinine to predict immunosuppression dosage

Problem framing

The mathematical model tracks compartments: susceptible cells (\(H_S\)), infected cells (\(H_I\)), BK virus (\(V\)), BK-specific T-cells (\(E_V\)), allospecific T-cells (\(E_K\)), and serum creatinine (\(C\)). The standard ODE equation for creatinine lacks mechanistic detail on the right-hand side:

\[ \dot{C}(t) = \lambda_C - \delta_{C0} \frac{H_S}{H_S + \kappa_{CH}} + h(t;\theta) \]

The function \(h(t;\theta)\) is unknown. The goal is to learn or modify this functional form through data-driven approaches, extracting an interpretable expression that clinicians can use to understand creatinine dynamics.

Clinical setting

Renal transplantation with serum-creatinine changes following BK virus infection. Data includes BKV load, creatinine, vitals, demographics, and dosage from Duke Hospitals EHR.

Interpretability target

Recover parsimonious governing terms from black-box PINN trajectories using symbolic regression, producing human-readable ODE terms.

Research outcome

Successfully extracted interpretable mathematical expressions for the missing physics of creatinine dynamics in BKV infection.

Three-Stage Pipeline

1

PINNs

Fit noisy longitudinal trajectories while penalizing violations of the candidate dynamical law, so the learned representation remains scientifically constrained.

2

Causal inference

Separate potentially meaningful drivers from spurious correlations to decide which variables and interactions deserve mechanistic interpretation.

3

Symbolic regression

Translate the learned black-box response into compact analytical terms that can be inspected, compared, and communicated as candidate ODE structure.

Framework schematic. Clinical data (C, V) feeds into an ODE model. PINNs learn the time-dependent RHS, causal inference identifies variables that affect h(t), and symbolic regression extracts the final interpretable form.

Compartment Model & Equations

Compartment model. Seven state variables govern BKV infection dynamics: susceptible cells (HS), infected cells (HI), BK virus (V), BK-specific T-cells (EV), allospecific T-cells (EK), serum creatinine (C), and hidden dynamics h.

Composite PINN loss

\[ \mathcal{L}_{\mathrm{PINN}} = \mathcal{L}_{\mathrm{data}} + \lambda_{\mathrm{phys}}\,\mathcal{L}_{\mathrm{physics}} \]

The data loss fits measured trajectories (creatinine, viral load). The physics loss penalizes violation of the compartment ODE system, enforcing biological consistency.

Creatinine equation with unknown RHS

\[ \dot{C}(t) = \lambda_C - \delta_{C0}\,\frac{H_S}{H_S + \kappa_{CH}} + h(t;\theta) \]

The function \(h(t;\theta)\) captures the unknown mechanism governing creatinine change. Causal inference identified that \(h\) depends on \(H_S\) and \(C\), and symbolic regression extracted an interpretable closed-form expression.
