Research Projects

Demand Forecasting with SHGP

Built a Sparse Hierarchical Gaussian Process (SHGP) model that captures both group-level trends across hotel chains and individual hotel behavior through a hierarchical kernel structure.

  • Custom SHGP model with group kernel (shared across chain) + individual kernels per hotel using GPflow
  • Predicted occupancy, booking-choice, and whole-group metrics across Accor, Choice, and Wyndham chains
  • Booking curve analysis to incorporate real-time reservation patterns
  • Compared forecasts against the IDeaS production system using days-to-arrival accuracy metrics
  • ARIMA and Naïve baselines for benchmark comparison

Special Event Detection with DMD

Designed a multi-method pipeline combining Dynamic Mode Decomposition, statistical anomaly detection, and GP classification to identify demand-driving events from occupancy time series.

  • Dynamic Mode Decomposition (DMD) to decompose occupancy into trend, seasonal, and residual components
  • Z-Score anomaly detection on residuals using same-day-of-week moving averages and standard deviations
  • GP regression with ordinal likelihood for event classification (high-volume vs. low-volume events)
  • Event calendar integration to correlate detected anomalies with known events (conferences, concerts, holidays, sports)
  • Gaussian Mixture Models (GMM) for unsupervised event clustering
